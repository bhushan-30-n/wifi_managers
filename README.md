A professional, feature-rich Wi-Fi management tool built in Python with a beautiful terminal interface.

![Wi-Fi Manager Demo](docs/demo.gif)

## ✨ Features

### Core Features
- 📋 **List Saved Profiles** - View all saved Wi-Fi networks
- 🔍 **View Passwords** - Retrieve saved Wi-Fi passwords (Admin required)
- 📡 **Current Connection** - Display current Wi-Fi status and signal strength
- 🔎 **Scan Networks** - Find available Wi-Fi networks in range
- 🗑️ **Delete Profiles** - Remove unwanted Wi-Fi profiles

### Advanced Features
- 💾 **Export Profiles** - Export to JSON, CSV, or TXT format
- 🛡️ **Security Analysis** - Check security score and vulnerabilities
- 📊 **Statistics** - View detailed network statistics
- 🔄 **Backup Profiles** - Create backups of your Wi-Fi profiles
- 📝 **Activity Log** - Track all actions performed
- 📶 **Signal Monitor** - Real-time signal strength monitoring

## 🚀 Installation

### Prerequisites
- Python 3.7 or higher
- Windows OS (for full functionality)
- Administrator privileges (for password viewing)

### Quick Install
```bash
# Clone the repository
git clone https://github.com/yourusername/wifi-manager.git
cd wifi-manager

# Install dependencies
pip install -r requirements.txt
n
